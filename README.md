# Customer Churn Data Analysis with Python

Data analysis project developed during **Jornada Python - Aula 2** focused on analyzing customer cancellation behavior and identifying factors that lead clients to cancel their subscriptions.

The goal of this project is to explore a dataset, perform data cleaning, and generate insights that can help a company reduce churn rates.

---

# Project Objective

The main objective of this project is to analyze a dataset of customers and identify patterns related to **subscription cancellations**.

Through data analysis we aim to:

- Understand the overall cancellation rate
- Identify factors that influence customer churn
- Explore relationships between variables
- Generate visual insights using graphs
- Propose possible strategies to reduce cancellations

---

# Dataset

The dataset simulates a company's customer base and contains information such as:

- Age
- Gender
- Time as customer
- Frequency of product usage
- Number of calls to call center
- Days of delay
- Subscription type
- Contract duration
- Total amount spent
- Months since last interaction
- Cancellation indicator
