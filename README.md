# 📊 Análise de Cancelamento de Clientes (Churn)

## 📌 Sobre o projeto

Este projeto simula um cenário real de negócio em uma empresa fictícia do segmento de serviços por assinatura, cujo objetivo é compreender os fatores que influenciam o cancelamento de clientes e apoiar a definição de estratégias de retenção.

Por meio de técnicas de análise exploratória de dados em Python, o estudo busca identificar padrões associados ao churn e gerar insights que contribuam para a redução da taxa de cancelamento.

---

## 🎯 Problema de negócio

Altas taxas de cancelamento impactam diretamente a receita e a sustentabilidade do negócio.

Neste contexto, o projeto analisa uma base de clientes para compreender:

- quais fatores influenciam o cancelamento  
- quais perfis possuem maior risco  
- quais ações podem ser adotadas para melhorar a retenção  

---

## 📊 Base de dados

A base contém informações como:

- idade  
- sexo  
- tempo como cliente  
- frequência de uso  
- número de ligações ao call center  
- dias em atraso  
- tipo de assinatura  
- duração do contrato  
- total gasto  
- meses desde a última interação  
- indicador de cancelamento  

---

## 🛠️ Etapas realizadas

- Importação e exploração inicial dos dados  
- Tratamento de valores ausentes  
- Análise da taxa de cancelamento  
- Análise por tipo de contrato  
- Análise por perfil de assinatura  
- Criação de visualizações para identificação de padrões  
- Filtragem de variáveis com maior impacto no churn  

---

## 📈 Principais insights

- Contratos mensais apresentaram maior taxa de cancelamento  
- Clientes com muitos dias de atraso possuem maior probabilidade de churn  
- Alta frequência de contato com o call center pode indicar insatisfação  
- Análises sucessivas permitiram reduzir a taxa simulada de churn para níveis mais aceitáveis
